# 🔐 Crypto System in C++

This project implements classical encryption and decryption algorithms:

## ✨ Algorithms Included:
- Caesar Cipher
- Rail Fence Cipher
- Vigenère Cipher
- Row Transposition Cipher

## ⚙️ Features:
- Encryption & Decryption
- Console-based menu system
- User-defined keys for each algorithm

## 🧪 How to Run:
Compile using any C++ compiler:

g++ main.cpp -o crypto
./crypto
